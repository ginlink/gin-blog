## 3 git stash

暂存

```sh
git stash save "test-cmd-stash"
```

恢复

```sh
# 弹出第一个(并删除)
git stash pop

# 查看stash列表
git stash list

# 运用指定stash(不删除)
git stash apply stash@{0}
```

[1] [`git stash`用法](https://www.cnblogs.com/tocy/p/git-stash-reference.html)
