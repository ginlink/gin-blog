## 1 typescript文件引入图片报错问题

需要新建一个ts声明文件：images.d.ts

```ts
declare module '*.svg'
declare module '*.png'
declare module '*.jpg'
declare module '*.jpeg'
declare module '*.gif'
declare module '*.bmp'
declare module '*.tiff'
```


然后将该images.d.ts文件路径配置到tsconfig.json中：

```ts
"include": [
    "./typings/images.d.ts" //文件路径
],
```
