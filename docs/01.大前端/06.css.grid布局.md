# Css.Grid

> 一定要做示例，勿眼高手低，常复习

思想：grid布局分为容器属性和项目属性

容器控制容器排列位置，还可以控制项目家的大小，在家中排列位置，还可以指定项目跨度

而项目可以指定自己的跨度，只占一个容器，还是夸容器占领多个容器

> 注意，容器是容器，项目是项目，元素是元素  它们都有各自的控制方法
