# react 全局监听路由（浏览器url）变化

通过useMemo+href即可，可以不用全局导航卫士，且react-router中**不**包含导航卫士

```ts
const href = location.href
useMemo(() => {
  // console.log('[123]:', href)

  changeIcon(href)
  // 根据地址去更改Tab的图标
}, [href])
```
