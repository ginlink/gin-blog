# css居中

### 4.1 块级元素居中

只说特殊的，垂直居中，方案有：1flex  2flex+margin【推荐】，水平居中都知道用margin

```css
.father {
    display: flex;
    /* 重点  必须设置为flex */
    min-height: 100vh;
    background: pink;
}

.son {
    margin: auto;
    /* 儿子用margin即可 */
    background: red;
}
```

联想：**为什么块级元素不能直接 `margin: auto 0;`来实现垂直居中呢？**

猜测和margin折叠有关，用 `display:flex;`之后子元素都是隔离的BFC，所以可以被margin撑起来
